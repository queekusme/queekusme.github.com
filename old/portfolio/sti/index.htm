Systematically Transmitted Infection

A Game developed for the 2018 Global Game Jam

Play as a virus, transmitting yourself across a network of computers, download the data from these computers to 'win'.

This gamer was created in 48 hours for the 2018 Global Game Jam.

Game code developed by myself in Unity. All of the game art assets were textured and modelled by myself using Cubik Studio and Paint Shop Pro.

Available on the Global Game Jam Website
